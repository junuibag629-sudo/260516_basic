# 깃허브를 이해하고 사용하기위한 첫 코드
### 1. 처음으로 깃허브의 사용법을 익히고 이해하기 위해 **계산기 코드**를 작성
#### 1) 일반적인 순서
- GitHub에서 새로운 Repository를 생성
- VS Code 터미널에서 프로젝트 폴더로 이동
- `git init`으로 Git 관리를 시작
- `git remote add origin 저장소URL`로 GitHub 저장소를 연결
- `git add .`로 변경된 파일을 스테이징
- `git commit -m "커밋 메시지"`로 변경 내용을 저장
- `git branch -M main` 후 `git push -u origin main`으로 최초 업로드
- 이후에는 코드 수정 → `git add .` → `git commit` → `git push` 순서로 작업
- 원격 저장소의 최신 코드를 받으려면 `git pull`을 사용
#### 2) Visual studio Code 실제 적용
- 코드를 작성
- git add . 으로 변경 파일을 등록
- git commit -m "수정 내용" 으로 버전을 생성
- git push 로 GitHub에 저장
- 이후 코드를 수정하고 다시 commit, push 하면 새로운 버전이 생성
- GitHub에서는 이전 버전과 현재 버전을 비교할 수 있음
- 필요하면 특정 버전으로 되돌리기 가능
- 따라서 코드 변경 이력을 체계적으로 관리 가능

### 2. 계산기 코드 작성 규칙
- 1회성이 아닌 반복하여 계산 진행
- 사칙연산 계산 기능과 계산기 종료 메뉴 포함
- 계산이 끝나거나 잘못 입력되면 다시 초기 메뉴 반복
- 0으로 나누면 계산 불가능 표시 후 다시 초기 메뉴 반복

### 3. 작성시 시행착오
- 처음 작성시 일단 쓰면서 생각하자는 마음으로 if문으로 시작하였으나 **작성 규칙에 반복 하라는 부분을 확인**
- **반복**을 하려면 while문을 써야 한다는 것이 생각나 **if문을 while문 안에 포함하는 구조로 변경**
- 작성을 대부분 완료하고 **반복은 성공**하였지만 **0을 눌러도 종료되지 않음**
- while문을 처음 사용할때 사용하는 **i = i + 1을 습관적으로 사용**한 탓에 0이되면 **종료해야하지만 반복하며 계속 1이 되는 문제**로 **끝나지 않던 문제를 수정**
### 4. 실제 구현 내용(검증 완료)
##### 1. 이 계산기는 1번 더하기 2번 빼기 3번 곱하기 4번 나누기 그리고 0번은 종료하는 메뉴 
##### 2. 메뉴에 5개의 메뉴 중에서 1,2,3,4번에서 한가지 숫자를 선택
##### 3. 네가지중 한 가지를 선택하면 다음은 두 가지 숫자를 적는 칸 생성  
##### 4. 두 가지 숫자를 받으면 처음에 선택한 번호에 저장된 사칙연산을 활용하여 계산을 완료
##### 5. 이때 4번인 나누기에 두 숫자를 적는 칸에 두번째 숫자를 0으로 적으면 나눌수 없다는 메세지가 나오며 다시 메뉴로 돌아감
##### 6. 그리고 다시 0~4까지 번호를 선택하는 메뉴로 돌아옴
##### 7. 마지막으로 메뉴에서 0번을 누르면 계산기가 종료

### 전체 코드 정리(필요한 부분 상세 주석 포함)
```
x = 1
while x != 0:
    print('==========사칙연산 계산기==========')
    x = int(input('1, 더하기\n2, 빼기\n3, 곱하기\n4, 나누기\n0, 종료\n  메뉴를 선택하세요:'))
    if x == 1:
        a,b = map(int,input('더하고 싶은 숫자 2개를 입력하시오:').split( ))
        print(a + b)
    elif x == 2:
        a,b = map(int,input('빼하고 싶은 숫자 2개를 입력하시오:').split( ))
        print(a - b)
    elif x == 3:
        a,b = map(int,input('곱하고 싶은 숫자 2개를 입력하시오:').split( ))
        print(a * b)
    elif x == 4:
        a,b = map(int,input('나누고 싶은 숫자 2개를 입력하시오:').split( ))
        if b == 0:
            print('0으로는 나눌 수 없습니다')
        else:
            print(a / b)
    elif x == 0:
        print(x, "종료")
        
    else:
        print('메뉴는 1, 2, 3, 4 밖에 없습니다 다시 선택하십쇼')
    

print('계산기 종료')
```
